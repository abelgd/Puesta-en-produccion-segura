# Puesta en Producción Segura  
### Tareas Individuales – Abel García Domínguez

## Descripción  
Repositorio correspondiente al módulo **Puesta en Producción Segura**, con prácticas enfocadas en la implementación segura de servicios, control de versiones y despliegue protegido de aplicaciones y sistemas.

## Contenido  
- Configuración de entornos de desarrollo y producción.  
- Aplicación de medidas de endurecimiento (hardening).  
- Gestión de actualizaciones y control de cambios.  
- Procedimientos de despliegue seguro y rollback.  

## Objetivo  
Desarrollar y aplicar buenas prácticas de seguridad en la transición de sistemas desde el entorno de desarrollo al de producción, asegurando la integridad, disponibilidad y confidencialidad de los activos tecnológicos.
